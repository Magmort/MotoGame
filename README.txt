/!\ IMPORTANT :
Pour lancer le jeu il faut ouvrir le fichier Index.html avec Firefox (Chrome bloque niveau sécurité pour la lecture des images)

COMMENT JOUER:
-Utilisez les touches directionnelles pour:
	HAUT 	: Accélérer
	BAS	: Freiner
	GAUCHE	: Tourner à Gauche
	DROITE	: Tourner à Droite
-Evitez les voitures pour ne pas perdre
-Evitez les flaques d'huiles sinon vous perdrez le contôle de la moto pendant une seconde
-Vous gagnez des points en dépassant les voitures celon la vitesse à laquelle vous les dépassées
-Dépasser des voitures en restant sur les bandes d'arrêt d'urgence ne rapporte pas de point mais peut vous sauver la vie

Amusez-vous bien,
